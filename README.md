# TaskFlow Frontend

A React TypeScript frontend for the TaskFlow task management application.

## Tech Stack

- **React 18** + **TypeScript**
- **Fetch API** — for REST API communication
- **CSS-in-JS** — inline styling

## Features

- View all tasks fetched from the backend
- Create tasks with title, description, and priority
- Mark tasks as complete / incomplete
- Delete tasks
- Color-coded priority labels (LOW / MEDIUM / HIGH)

## Getting Started

### Prerequisites
- Node.js 16+
- TaskFlow Backend running on `http://localhost:8080`

### Run the App
```bash
npm install
npm start
```
App runs on `http://localhost:3000`
